首先在git中运行以克隆仓库
```git
git clone https://github.com/Skrepy0/xuetangzaixian-crawler.git
```
<Picture
url="/assets/articles/tutorial/20260611/1.webp"
/>

然后在cmd中运行
```bash
cd xuetangzaixian-crawler
```
进入仓库根目录
创建python虚拟环境并激活
```bash
python -m venv venv
# Windows
venv\Scripts\Activate.ps1
# macOS/Linux
source venv/bin/activate
```
此时根目录多出venv文件夹

然后运行下面的命令安装依赖
```bash
pip install -r requirements.txt
```

然后就是补全main.py中的参数
```python
cookies = {
    # your cookie here
}
class_front_code = '' # 课程编号 
class_code = '' # 班级编号 
code_start = 20 # 起始章节
code_end = 100 # 结束章节
```

首先打开学堂在线网站并进入你要爬取的课程学习界面(就是看视频的界面)
然后在左侧目录部分从第一章开始找,找到第一个单元测试界面
<Picture
url="/assets/articles/tutorial/20260611/2.webp"
height = 320
/>

然后按`F12`,并点击"网络"
<Picture
url="/assets/articles/tutorial/20260611/3.webp"
/>

然后点击`F5`刷新界面
在"名称"中找到类似下面这种请求,并点击


<Picture
url="/assets/articles/tutorial/20260611/4.webp"
/>

然后点击"标头"
<Picture
url="/assets/articles/tutorial/20260611/5.webp"
/>

请求网址例如:
```
https://www.xuetangx.com/**/6611528/14925761/
```

然后获取其中`6611528`号码,'66115'填入main.py中的class_front_code
28为code_start的值

最后一个号码`14925761`为班级编号,即class_code

同理,找到目录中最后一个练习,获取code_end的值

示例配置
```python
class_front_code = '66115'
class_code = '14925761' 
code_start = 28 
code_end = 100
```

同时请求向下翻,找到`cookie`,并复制其右方的值

<Picture
url="/assets/articles/tutorial/20260611/6.webp"
height=500
info="本图最后的键"
/>

根目录下运行`python cookies_to_json.py`,
将你复制的cookie值粘贴进去,并回车,得到json格式的cookie,将它粘贴到main.py的cookie配置中

```python
cookies = {
  "_abfpc": "--",
  "cna": "--",
  "mode_type": "--",
  ...
}
```

然后运行爬虫
```bash
python main.py
```
输出在dist目录下
